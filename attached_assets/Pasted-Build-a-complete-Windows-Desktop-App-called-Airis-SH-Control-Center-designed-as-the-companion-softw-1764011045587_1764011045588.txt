Build a complete Windows Desktop App called Airis-SH Control Center, designed as the companion software for a futuristic gesture-based air mouse for autistic and physically-disabled individuals.

The app must be visually modern, minimal, soft, rounded, and accessible.
Use soft gradients, glass-morphism panels, large cards, animated indicators, gentle shadows, and smooth transitions.

Primary colors: sky blue, white, soft teal, pastel gradients.
Primary style: calming, friendly, non-technical, disability-friendly.

APP STRUCTURE (Build all screens)
1. Dashboard Screen

Bluetooth connection status (Connected / Searching / Not Connected)

Wired mode status

Battery percentage (with circular animated gauge)

Current Mode: "Normal", "Comfort Mode", "Safe Mode"

Live gesture indicator (glowing dot or waveform animation)

3 LED indicators preview dots (matching device look)

2. Gesture Settings Screen

Adjustable Left Click dwell time (slider 1–3 seconds)

Adjustable Right Click twist sensitivity (slider)

Toggle: “Enable SOS Gesture”

Toggle: “Enable Auto-Center Gesture”

Toggle: “Enable Volume Twist Gesture” (optional)

Live preview animation showing the gesture

Reset to defaults button

3. Comfort Mode Screen

Toggle switch: ON/OFF

Anti-shake strength slider

Cursor smoothing level slider

LED calming animation preview

Information card explaining Comfort Mode for autistic users

4. Accessibility Tools Screen

Toggle: “Auto Open On-Screen Keyboard when typing is needed”

Toggle: “Highlight cursor with glow ring”

Toggle: “Slow Mode for unsteady hands”

Toggle: “Large pointer mode”

Toggle: “Disable clicks temporarily (Safe Mode)”

5. Caregiver / Emergency Screen

Add emergency contact (Name, Phone, Email)

Message template for emergency alerts

Trigger test SOS

Status indicator: “SOS Gesture Active”

6. Device Settings Screen

Sensor calibration wizard for MPU6050

Sensitivity sliders for X/Y motion

Firmware update upload button

Factory reset card

Device info block: Name = Airis-SH, Version, Serial

7. About Screen

Story behind the product

Designed for autistic & disabled users

Credits

Small animation or logo of Airis-SH
